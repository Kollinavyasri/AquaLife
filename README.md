# 🌊 AquaLife – Smart Aquatic Ecosystem Monitoring

AquaLife is a web-based environmental monitoring dashboard that analyzes river ecosystem health using environmental and water quality indicators.

The system visualizes river data, evaluates ecosystem health, and provides suggestions for maintaining aquatic balance.

---

## 📌 Project Overview

Water ecosystems are vital for biodiversity, agriculture, and human life. Pollution and environmental changes affect rivers and aquatic organisms.

**AquaLife** analyzes environmental data and provides insights into river health using a simple interactive dashboard.

The system evaluates factors such as:

* Water pH
* Dissolved Oxygen (DO)
* Biochemical Oxygen Demand (BOD)
* PM2.5 pollution levels
* Temperature
* Humidity

## 🌍 Ecosystem Health Logic

The system evaluates water health using environmental thresholds:

| Parameter        | Healthy Range |
| ---------------- | ------------- |
| pH               | 6.5 – 8.5     |
| Dissolved Oxygen | > 6 mg/L      |
| BOD              | < 3 mg/L      |
| PM2.5            | < 60          |
| Temperature      | < 30°C        |

Based on these values, the ecosystem is classified as:

* 🟢 Healthy
* 🟡 Moderate
* 🔴 Critical

---

## 💡 Suggestions System

AquaLife provides suggestions when water quality declines.

Examples:

* Reduce pollution discharge
* Protect aquatic plants and fish
* Monitor irrigation water quality
* Maintain clean river surroundings
